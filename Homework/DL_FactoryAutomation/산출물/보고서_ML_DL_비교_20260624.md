# CNC 설비 고장 예측 - DL 도입 성능 개선 보고서 (요약)

작성일: 2026-06-24  담당: 허남

---

## 핵심 요약

> DL(딥러닝)은 ML(머신러닝)이 처리하지 못하는 시계열/원시신호 데이터를
> 추가로 활용하여 CNC 설비 고장 탐지 범위와 정확도를 함께 개선합니다.

| 구분 | 모델 | 데이터 | F1(고장) | 특징 |
|------|------|--------|---------|------|
| Source 1 (ML) | XGBoost | AI4I 정형 10K행 | 0.724 | 빠른 단일값 판단 |
| Source 2 (DL) | CNN+LSTM | 합성 시계열 60K | 학습 중 | 50스텝 열화 궤적 |
| Source 3-ML | RandomForest | NASA Milling | 학습 중 | 수동 FFT 피처 |
| Source 3-DL | 1D-CNN | NASA Milling | 학습 중 | 원시신호 직접 처리 |

---

## 3-Source 계층 아키텍처

  [Layer 1 - ML]   정형 스냅샷 -> XGBoost    -> F1 0.724  (기준선)
  [Layer 2 - DL]   시계열 50스텝 -> CNN+LSTM  -> 열화 조기 감지
  [Layer 3 - DL]   원시신호 512샘플 -> 1D-CNN  -> 피처공학 없이 직접 분류

## ML 대비 DL 핵심 우위

| 항목 | ML(XGBoost) | DL(CNN+LSTM, 1D-CNN) |
|------|------------|----------------------|
| 입력 형태 | 단일 스냅샷 | 시계열 / 원시파형 |
| 특성공학 | 도메인 전문가 필요 | 자동 학습 |
| 열화 패턴 감지 | 불가 | 50스텝 궤적으로 포착 |
| 고주파 신호 처리 | FFT 수동 계산 | Conv1D 자동 추출 |
| 조기 경보 | 불가(현재값만) | 가능(추세 학습) |

## 현장 적용 타당성 (Reasonable & Makes Sense)

| 계층 | 역할 | 응답시간 | 현장 사례 |
|------|------|---------|---------|
| Layer 1 ML | 실시간 알람 | 1ms 미만 | SCADA 모든 공장 |
| Layer 2 DL | 조기 경보 | 50ms 미만 | Siemens MindSphere |
| Layer 3 DL | 정밀 진단 | 200ms 미만 | 항공/자동차 정밀가공 |
